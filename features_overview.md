# 功能梳理（更新版）

## 1. 核心玩法循环
- 资源链路：碎片 -> 零件 -> 蓝图 -> 定律 -> 常数。
- 生产与转化：采集、转换、合成、能量分配与效率影响。
- 自动定律：蓝图达到阈值自动生成定律。
- 运行变体：每轮随机词条影响产能与效率。

## 2. 建筑与购买
- 建筑类型：碎片生产、碎片转零件、零件转蓝图、能量生产。
- 购买支持：买1/买10/买最大，成本按指数增长。
- 购买反馈：建筑卡片轻量缩放反馈 + 粒子飞入资源条。

## 3. 研究系统
- 三条分支：工业、算法、宇宙。
- 研究前置与成本（蓝图），解锁后提供永久增益。
- 研究详情弹出层：效果、成本、前置、购买按钮。
- 新增多层研究节点（工业/算法/宇宙追加阶段）。

## 4. 里程碑系统
- 达成条件触发加成。
- 里程碑展示与达成状态。

## 5. 升维与常数
- 升维预估：基于定律计算常数。
- 升维重置：资源/设施/配比/研究重置，保留常数与里程碑。
- 升维过场：轻量级缩放/扭曲动画。
- 常数强化：产出倍率、速度倍率、离线时长上限。
- 挑战升维：独立挑战目标、永久解锁奖励，双重确认后开启，开启后不可取消。

## 6. 技能系统
- 技能树：主动/被动技能，使用蓝图或技能点解锁。
- 技能装配：最多 5 个技能槽，装配后生效。
- 主动技能：时间扭曲/手动超频/资源脉冲，带冷却与持续时间。
- 快捷栏：跨页面一键启用主动技能。

## 7. 设施布局与协同
- 设施布局：研究解锁后开启布局与邻接加成。
- 布局格位：当前 5x4，共 20 格。
- 邻接协同：相邻设施产生额外产能与效率加成。

## 8. 离线收益
- 进入游戏时补算离线收益，受离线时长上限影响。
- 离线收益日志与弹窗提示。
- 研究解锁后支持离线合成蓝图。

## 9. UI 与布局
- 顶部资源条：支持超长数值适配，异常提示闪烁。
- 底部导航：生产 / 研究 / 技能 / 升维 / 日志。
- 生产页：产能概览、变体卡片、定律进度、设施卡片、配比与布局面板。
- 研究页：分支筛选、节点网格、弹出购买面板。
- 技能页：技能树节点、详情面板、装配与强化。
- 升维页：预估、重置/保留说明、常数强化、挑战与永久解锁。
- 日志页：事件记录、里程碑、存档导入/导出。

## 10. 存档与导入导出
- 本地存档：Hive 持久化，自动迁移旧存档。
- 导出/导入：JSON 文本复制与粘贴。

## 11. 主题与视觉
- 深色太空背景与网格纹理。
- 统一字体与卡片风格。
- 粒子与升维过场动效。

## 12. 其他

- 全局中文文案。
