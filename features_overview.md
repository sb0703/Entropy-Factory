# 功能梳理

## 1. 核心玩法循环
- 资源链路：碎片 -> 零件 -> 蓝图 -> 定律 -> 常数。
- 生产与转化：采集、转换、合成、能量分配与效率影响。
- 自动定律：蓝图达到阈值自动生成定律。

## 2. 建筑与购买
- 建筑类型：碎片生产、碎片转零件、零件转蓝图、能量生产。
- 购买支持：买1/买10/买最大，成本按指数增长。
- 购买反馈：建筑卡片轻量缩放反馈。

## 3. 研究系统
- 三条分支：工业、算法、宇宙。
- 研究前置与成本（蓝图），解锁后提供永久增益。
- 研究详情面板：效果、成本、前置、购买按钮。

## 4. 里程碑系统
- 达成条件触发加成。
- 里程碑展示与达成状态。

## 5. 升维与常数
- 升维预估：基于定律计算常数。
- 升维重置：资源/设施/配比/研究重置，保留常数与里程碑。
- 升维过场：轻量级缩放/扭曲动画。
- 常数强化：产出倍率、速度倍率、离线时长上限。

## 6. 离线收益
- 进入游戏时补算离线收益，受离线时长上限影响。
- 离线收益日志与弹窗提示。
- 研究解锁后支持离线合成蓝图。

## 7. UI 与布局
- 顶部资源条，底部四页导航：生产 / 研究 / 升维 / 日志。
- 生产页：产能概览、定律进度、设施卡片、配比面板。
- 研究页：分支筛选、节点网格、详情面板。
- 升维页：预估、重置/保留说明、常数强化面板。
- 日志页：事件记录、里程碑、存档导入/导出。

## 8. 存档与导入导出
- 本地存档：shared_preferences 持久化。
- 导出/导入：JSON 文本复制与粘贴。

## 9. 主题与视觉
- 深色太空背景与网格纹理。
- 统一字体与卡片风格。

## 10. 其他
- 数值格式化（万/亿）。
- 全局中文文案。
